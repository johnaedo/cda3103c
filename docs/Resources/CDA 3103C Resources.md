# Resources

This is a list of helpful resources for learning CDA 3103C content, contributed to by many students and Professor Aedo, compiled by sh0inx.

## How to Use This
Generally speaking, you want to use these as **supplemental** resources to the textbook and class notes. While the textbook is not *required*, it is *optional*, and provides much of the background and context in which the class is taught.

Get a good night's rest, hit the books when you're ready. If you're unsure how you want to study, [here are some helpful tips](https://learningcenter.unc.edu/tips-and-tools/studying-101-study-smarter-not-harder/), but most importantly, do what works for you and doesn't make you hate the material.
